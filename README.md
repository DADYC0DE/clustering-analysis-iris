# Clustering Analysis - Iris Dataset

## Objective
To evaluate clustering algorithms using performance metrics and visualization.

## Algorithms Used
- K-Means
- Agglomerative Clustering
- DBSCAN

## Evaluation Metrics
- Inertia
- Silhouette Score
- Calinski-Harabasz Index
- Davies-Bouldin Index

## Results
- Optimal clusters: 3
- K-Means performed best

## Tools
- Python
- Scikit-learn
- Matplotlib
